# **App Name**: QuizLive

## Core Features:

- Quiz Creation: Enable hosts to create quizzes with multiple-choice questions, set time limits, and define correct answers using a streamlined interface.
- Live Game Hosting: Provide a host interface to launch quizzes, manage game progression, and display questions and answer options in real-time.
- Player Participation: Allow players to join games using a unique game code, submit answers via an intuitive interface, and view immediate feedback on answer correctness.
- Real-time Leaderboard: Display a live leaderboard showcasing player rankings based on scores and response time, fostering a competitive and engaging environment.
- AI-Powered Question Suggestion: Suggest questions or topics based on the category provided by the tool

## Style Guidelines:

- Primary color: Vibrant purple (#9C27B0) to convey energy and playfulness.
- Background color: Light lavender (#F3E5F5) for a soft, inviting backdrop.
- Accent color: Electric blue (#2962FF) to highlight interactive elements and calls to action.
- Body and headline font: 'Space Grotesk' sans-serif for a modern, techy appearance.
- Use bright, geometric icons representing quiz categories and game actions.
- Design a clean, responsive layout optimized for both host and player interfaces, ensuring readability and ease of navigation.
- Incorporate subtle animations for correct answers, leaderboard updates, and game transitions to enhance user engagement.
